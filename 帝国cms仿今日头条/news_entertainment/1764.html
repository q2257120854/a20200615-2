    超过30万位的宋太太已疯！为太阳后裔原声泪奔～

    2016-04-20 20:09

    韩国大型虐狗偶像剧《太阳的后裔》虽然在上周大结局了，但很多“性情中人”至今仍沉浸在虐心虐肺的情节里不能自拔，甚至有些宋太太们又开始从头看起继续舔屏。其实，值得回味的，除了宋老公的颜值，单身狗教科书一样的“撩妹”情节外，还有在剧中渲染气氛、升华情感的背景音乐。

    《太阳的后裔》里的背景音乐，光听旋律就已经超级打动人，但如果你听懂了这些歌，里面的一字一句真的会让你瞬间泪崩。

    姜暮烟：我不敢爱你 但我爱你

    无数次拒绝，又无数次欲言又止。我爱你，我却不敢爱你，我不敢爱你，但我又很爱你。那简单的一句“我爱你”，几乎用尽了姜暮烟毕生的力气。

    “我曾经无法开始的爱情 / 现在可以说出口了 / 这可是谁都无法拥有的奇迹呢 / You are my everything / 我满腔热血的爱恋是你 / 即使季节更替我也会在原地 / 我停驻不前的内心 / 我唯一的爱 / You are my everything―― 《You Are My Everything》Gummy ”

    柳时镇：我爱你 可不可以别再推开我

    对于柳时镇的每一次走近，姜暮烟总是胆怯地后退。她不是不爱他，只是怕在茫茫未知的某天，再也不能爱他。柳时镇：姜暮烟，我爱你，但你可不可以别再推开我？

    “How can I stay with you / 试着闭上眼睛我就在你身边 / 为了让我能够触碰到你的双颊 / 可以向我走近一步吗――《How Can I Love You》金俊秀”

    尹明珠：一直相爱吧 不要再躲了

    姜暮烟：徐大荣的工作那么危险，你就不担心他牺牲吗？

    尹明珠：对我来说，见不到他，比死亡更可怕

    “如果再也见不到你 / 对我来说 / 比死亡还要可怕 / 留在我身边吧 / 不要离开我 / 你是我唯一的爱 / 即使把一生奉献给你 / 都不会感到惋惜 / 我们爱得如此深刻 / 伤得痛彻心扉 / 情愿 情愿 情愿 就这样 / 一直相爱吧――《让我们相爱》SG Wannabe ”

    徐大荣：对不起 我爱你

    因二人身份悬殊，深爱尹明珠的徐大荣总以为放手才是对明珠最好的爱。不断地逃避这份感情他，直到明珠遇险才猛然醒悟：抓住你的手不放，才是真正爱你的方式。

    “如果能再来一次的话 / 我会试图抓住你的手 / 绝对不会放开 / 你只需要美丽地绽放就好 / 我会为了你成为荆棘 / 该死的 / 为什么我没能想到呢 / 那时候能抓住的话 / 现在就不同了吧――《再次见到你》Mad Clown&金娜英”

    看懂了这些歌词之后，是不是更想重温这10首虐得我们一脸血的OST呢！什么？你竟然不知道在哪里听？！《太阳的后裔》十首正版原声音乐，全都在酷狗音乐独家发布的哦。自《太阳的后裔》OST在酷狗平台上架以来，便受到众多宋太太们的狂热追捧，专辑销量至今已突破30W张！也就是说，已经有30多万人通过酷狗音乐珍藏这张专辑啦，这人气真真杠杠滴！

    《太阳的后裔》10首OST完整版现已全部收录完毕，深爱宋老公的你又怎能错过？赶紧来酷狗畅听吧！
